# Overlay Notepad

Лёгкий оверлей для Windows, который всегда остаётся поверх остальных окон и показывает пары «ключ – значение» из текстового файла. Виджет удобно закрепляется в произвольной области экрана, умеет мгновенно фильтровать записи и копирует значения по клику.

[⬇️ Скачать готовый билд](https://github.com/TimaSkye/myproject_overlay_notepad/raw/refs/heads/master/OverlayNotepad.exe)

</div>

## Возможности

- **Всегда поверх окон** — используется набор Qt-флагов и слежение за перемещением, чтобы плашка не теряла фокус.
- **Мгновенный поиск** — наведите курсор на виджет и начните печатать. `Backspace` удаляет символ, `Esc` очищает фильтр.
- **Файл наблюдается автоматически** — `QFileSystemWatcher` перечитывает `file.txt` при сохранении, данные обновляются без перезапуска.
- **Удобные настройки** — диалог позволяет подобрать шрифт, размеры, цвета, прозрачность, файл данных и включить автозапуск.
- **Системный трэй** — иконка позволяет быстро открыть настройки, файл или информацию «О программе».

## Быстрый старт

1. Убедитесь, что установлен Python 3.12+ и [uv](https://docs.astral.sh/uv/) (например, `pipx install uv`).
2. Клонируйте репозиторий и установите зависимости:

   ```bash
   uv pip install -e .
   ```

3. Запустите приложение:

   ```bash
   uv run python main.py
   ```

   Первый запуск покажет мастер настроек. Далее будет использоваться `settings.json` из корня (можно переопределить ключом `--config-file`).

### Аргументы командной строки

| Параметр | Описание |
| --- | --- |
| `--file PATH` | Источник данных (по умолчанию `file.txt` в корне). |
| `--config-file PATH` | Альтернативный конфиг c настройками UI. |
| `--setup` | Принудительно открыть диалог настройки перед запуском. |

## Формат данных

`file.txt` — это набор JSON-объектов по одному на строку, каждый содержит единственную пару ключ/значение:

```json
{"VPN пароль": "SeCr3t!"}
{"GitHub": "github.com/TimaSkye"}
```

Ключ используется для поиска и отображается слева, значение копируется в буфер обмена по клику.

## Сборка и релизы

- Линтинг: `uv run ruff check .`
- Запуск в режиме разработки: `uv run python main.py --setup`
- Сборка standalone-билда:

  ```bash
  uv run pyinstaller OverlayNotepad.spec
  ```

  Готовый исполняемый файл лежит в `dist/OverlayNotepad.exe`. Для публикации он копируется в корень репозитория (`OverlayNotepad.exe`) и добавлен в Белый список `.gitignore`.

## Структура проекта

```
├─ main.py                    # точка входа из репозитория
├─ OverlayNotepad.spec        # конфигурация PyInstaller
├─ OverlayNotepad.exe         # актуальный собранный билд
├─ src/overlay_notepad/
│   ├─ app.py                 # QApplication, системный трей, слежение за файлом
│   ├─ data_store.py          # чтение и фильтрация JSON-строк
│   ├─ models.py              # вспомогательные структуры данных
│   ├─ settings.py            # менеджер конфигурации
│   └─ ui/                    # все Qt-виджеты: overlay, диалоги, селектор области
└─ pyproject.toml             # зависимости и метаданные для uv
```

`file.txt` и `settings.json` создаются пользователем: первый — это ваш словарь пар, второй появляется автоматически после сохранения настроек.
